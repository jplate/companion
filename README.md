# companion
An interactive 'companion' for some papers I authored (bootstrapped with Claude Fable 5)

Initial prompt:
I'd like you to create a web app based on my most important published works, which I've uploaded. It should respond to user queries of, e.g., the form, 'Explain in simple terms how the independence assumption in §5.4 of "Ordinal Type Theory" works'. The web app should be user friendly and nice to look at, and it should act, in a subtle way, as a showcase for my work.
// end of prompt //

The papers in question are:
- 'Logically Simple Properties and Relations', Philosophers' Imprint (2016)
- 'Intrinsic Properties and Relations', Inquiry (2018)
- 'Qualitative Properties and Relations', Philosophical Studies (2022)
- 'Quo Vadis, Metaphysics of Relations?', Dialectica (2022; introduction to a special issue)
- 'Ordinal Type Theory', Inquiry (2025)
- 'Towards an Ontology of Roles and States', Erkenntnis (forthcoming; published online 2026)
